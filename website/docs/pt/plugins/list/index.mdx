# Lista de Plugins

## Sistema de Plugins

Você pode ler sobre a funcionalidade dos plugins do Rsbuild e como desenvolver um plugin Rsbuild na documentação [Sistema de Plugins](/plugins/dev/index).

## Usando plugins

Você pode registrar plugins Rsbuild no arquivo `rsbuild.config.ts` usando a opção `plugins`. Para mais detalhes, consulte [plugins](/config/plugins).

Se você estiver usando a API JavaScript do Rsbuild, você pode registrar o plugin usando o método [addPlugins](/api/javascript-api/instance#rsbuildaddplugins).

## Plugins oficiais

A seguir estão os plugins oficiais que podem ser usados no Rsbuild.

### React

Plugins disponíveis para o React:

- [@rsbuild/plugin-react](/plugins/list/plugin-react): Fornece suporte para React.
- [@rsbuild/plugin-svgr](/plugins/list/plugin-svgr): Suporta a conversão de SVG para componentes React.
- [@rsbuild/plugin-styled-components](https://github.com/rsbuild-contrib/rsbuild-plugin-styled-components): Fornece suporte em tempo de compilação para styled-components.

### Vue

Plugins disponíveis para o Vue:

- [@rsbuild/plugin-vue](/plugins/list/plugin-vue): Fornece suporte para Vue 3 SFC (Single File Components).
- [@rsbuild/plugin-vue-jsx](https://github.com/rspack-contrib/rsbuild-plugin-vue-jsx): Fornece suporte para sintaxe Vue 3 JSX / TSX.
- [@rsbuild/plugin-vue2](https://github.com/rspack-contrib/rsbuild-plugin-vue2): Fornece suporte para Vue 2 SFC (Single File Components).
- [@rsbuild/plugin-vue2-jsx](https://github.com/rspack-contrib/rsbuild-plugin-vue2-jsx): Fornece suporte para sintaxe Vue 2 JSX / TSX.

### Preact

Plugins disponíveis para o Preact:

- [@rsbuild/plugin-preact](/plugins/list/plugin-preact): Fornece suporte para Preact.

### Svelte

Plugins disponíveis para o Svelte:

- [@rsbuild/plugin-svelte](/plugins/list/plugin-svelte): Fornece suporte para componentes Svelte (arquivos `.svelte`).

### Solid

Plugins disponíveis para o Solid:

- [@rsbuild/plugin-solid](/plugins/list/plugin-solid): Fornece suporte para Solid.

### Comum

Os seguintes são plugins comuns agnósticos de framework:

- [@rsbuild/plugin-assets-retry](https://github.com/rspack-contrib/rsbuild-plugin-assets-retry): Usado para reenviar automaticamente requisições quando ativos estáticos falham ao carregar.
- [@rsbuild/plugin-babel](/plugins/list/plugin-babel): Fornece suporte para capacidades de transpilação Babel.
- [@rsbuild/plugin-sass](/plugins/list/plugin-sass): Usa Sass como pré-processador CSS.
- [@rsbuild/plugin-less](/plugins/list/plugin-less): Usa Less como pré-processador CSS.
- [@rsbuild/plugin-stylus](/plugins/list/plugin-stylus): Usa Stylus como pré-processador CSS.
- [@rsbuild/plugin-basic-ssl](https://github.com/rspack-contrib/rsbuild-plugin-basic-ssl): Gera um certificado autoassinado não confiável para o servidor HTTPS.
- [@rsbuild/plugin-eslint](https://github.com/rspack-contrib/rsbuild-plugin-eslint): Executa verificações ESLint durante a compilação.
- [@rsbuild/plugin-type-check](https://github.com/rspack-contrib/rsbuild-plugin-type-check): Executa o verificador de tipo TypeScript em um processo separado.
- [@rsbuild/plugin-image-compress](https://github.com/rspack-contrib/rsbuild-plugin-image-compress): Comprime os ativos de imagem.
- [@rsbuild/plugin-mdx](https://github.com/rspack-contrib/rsbuild-plugin-mdx): Fornece suporte para MDX.
- [@rsbuild/plugin-node-polyfill](https://github.com/rspack-contrib/rsbuild-plugin-node-polyfill): Usado para injetar polyfills de módulos principais do Node no lado do navegador.
- [@rsbuild/plugin-source-build](https://github.com/rspack-contrib/rsbuild-plugin-source-build): Este plugin é projetado para o cenário monorepo. Ele suporta a referência de código-fonte de outros subdiretórios e executa a build e a atualização a quente.
- [@rsbuild/plugin-check-syntax](https://github.com/rspack-contrib/rsbuild-plugin-check-syntax): Verifica a compatibilidade de sintaxe dos arquivos de saída e determina se há alguma sintaxe avançada que possa causar problemas de compatibilidade.
- [@rsbuild/plugin-css-minimizer](https://github.com/rspack-contrib/rsbuild-plugin-css-minimizer): Usado para personalizar o minificador CSS, alternar para [cssnano](https://github.com/cssnano/cssnano) ou outras ferramentas para compressão CSS.
- [@rsbuild/plugin-typed-css-modules](https://github.com/rspack-contrib/rsbuild-plugin-typed-css-modules): Gera arquivo de declaração TypeScript para CSS Modules.
- [@rsbuild/plugin-pug](https://github.com/rspack-contrib/rsbuild-plugin-pug): Fornece suporte para o motor de template Pug.
- [@rsbuild/plugin-rem](https://github.com/rspack-contrib/rsbuild-plugin-rem): Implementa o layout adaptativo rem para páginas móveis.
- [@rsbuild/plugin-umd](https://github.com/rspack-contrib/rsbuild-plugin-umd): Gera saídas no formato UMD.
- [@rsbuild/plugin-yaml](https://github.com/rspack-contrib/rsbuild-plugin-yaml): Importa arquivos YAML e os converte em objetos JavaScript.
- [@rsbuild/plugin-toml](https://github.com/rspack-contrib/rsbuild-plugin-toml): Importa arquivos TOML e os converte em objetos JavaScript.

:::tip
Você pode encontrar o código-fonte de todos os plugins oficiais em [web-infra-dev/rsbuild](https://github.com/web-infra-dev/rsbuild) e [rspack-contrib](https://github.com/rspack-contrib).
:::
