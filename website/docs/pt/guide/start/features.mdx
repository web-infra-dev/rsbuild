# Recursos

Aqui estão todos os principais recursos suportados pelo Rsbuild.

## JavaScript

| Recursos                       | Descrição                                                                                                                                  | Links                                                                                                                                      |
| :----------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| Rspack                         | Usa o Rspack como o empacotador por padrão                                                                                                 | <ul><li>[Configurar Rspack](/guide/configuration/rspack)</li></ul>                                                                         |
| Compilação SWC                 | Transforma e minifica o código JavaScript e TypeScript com SWC por padrão                                                                  | <ul><li>[Configurar SWC](/guide/configuration/swc)</li></ul>                                                                               |
| Compilação TS                  | Arquivos TS são compilados pelo SWC por padrão                                                                                             | <ul><li>[Compilação TypeScript](/guide/basic/typescript#typescript-compilation)</li></ul>                                                  |
| Minificação de Código          | A minificação de código é ativada por padrão no modo de produção                                                                           | <ul><li>[output.minify](/config/output/minify)</li></ul>                                                                                   |
| Injeção de Polyfill            | Recurso opcional, injeta core-js e outros polyfills                                                                                        | <ul><li>[Compatibilidade do Navegador](/guide/advanced/browser-compatibility)</li><li>[output.polyfill](/config/output/polyfill)</li></ul> |
| Geração de SourceMap           | Mapas de origem são gerados no modo de desenvolvimento por padrão                                                                          | <ul><li>[output.sourceMap](/config/output/source-map)</li></ul>                                                                            |
| Alias                          | Recurso opcional, define o alias de importação                                                                                             | <ul><li>[resolve.alias](/config/resolve/alias)</li></ul>                                                                                   |
| Compilação Babel               | Recurso opcional, usa o Babel para transformar o código JavaScript e TypeScript                                                            | <ul><li>[Plugin Babel](/plugins/list/plugin-babel)</li></ul>                                                                               |
| Saídas Node                    | Recurso opcional, construindo pacotes que rodam no ambiente Node.js                                                                        | <ul><li>[Alvo Node](/config/output/target#node-target)</li></ul>                                                                           |
| Web Workers                    | Recurso opcional, usa Web Workers                                                                                                          | <ul><li>[Web Workers](/guide/basic/web-workers)</li></ul>                                                                                  |
| Browserslist                   | Recurso opcional, usando browserslist (abre uma nova janela) para especificar quais navegadores devem ser suportados em sua aplicação web. | <ul><li>[Browserslist](/guide/advanced/browserslist)</li></ul>                                                                             |
| Verificação de Compatibilidade | Recurso opcional, analisa se há sintaxe avançada incompatível nas saídas dentro do escopo do navegador atual                               | <ul><li>[@rsbuild/plugin-check-syntax](https://github.com/rspack-contrib/rsbuild-plugin-check-syntax)</li></ul>                            |
| Variável de Ambiente           | Recurso opcional, injeta variáveis de ambiente ou expressões no código                                                                     | <ul><li>[Variáveis de ambiente](/guide/advanced/env-vars)</li></ul>                                                                        |
| Polyfill de Node               | Recurso opcional, injeta polyfills para módulos principais do Node no lado do navegador                                                    | <ul><li>[Plugin de polyfill de Node](https://github.com/rspack-contrib/rsbuild-plugin-node-polyfill)</li></ul>                             |
| Verificação de Tipo            | Recurso opcional, executa o verificador de tipo para verificar problemas de tipo no código                                                 | <ul><li>[Verificação de tipo](/guide/basic/typescript#type-checking)</li></ul>                                                             |
| Module Federation              | Recurso opcional, carrega módulos dinamicamente e compartilha dependências                                                                 | <ul><li>[Module Federation](/guide/advanced/module-federation)</li></ul>                                                                   |
