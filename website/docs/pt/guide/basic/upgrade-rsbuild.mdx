# Atualizar Rsbuild

Esta seção explica como atualizar as dependências do Rsbuild do projeto para a versão mais recente.

:::tip

- Consulte [Lançamentos](/community/releases/index) para saber mais sobre a estratégia de lançamento do Rsbuild.
- Consulte [npm - @rsbuild/core](https://npmjs.com/package/@rsbuild/core) para visualizar a versão mais recente.

:::

## Usando taze

Recomendamos usar [Taze](https://github.com/antfu-collective/taze) para atualizar a versão do Rsbuild. Taze é uma ferramenta CLI para atualizar dependências npm.

### Uso

Execute o seguinte comando para atualizar todas as dependências que incluem `rsbuild` em seus nomes:

```bash
npx taze --include /rsbuild/ -w
```

O resultado será semelhante a:

```bash
rsbuild - 3 patch

  @rsbuild/core               dev  ~1mo  ^1.0.0  →  ^1.2.0
  @rsbuild/plugin-react       dev  ~1mo  ^1.0.0  →  ^1.2.0
  @rsbuild/plugin-type-check  dev  ~1mo  ^1.0.0  →  ^1.2.0

ℹ changes written to package.json, run npm i to install updates.
```

Você também pode ajustar o padrão `include` para corresponder a pacotes específicos, por exemplo, para atualizar apenas pacotes sob o escopo `@rsbuild`:

```bash
npx taze --include /@rsbuild/ -w
```

### Opções

Aqui estão alguns exemplos de uso das opções do taze.

- Em um monorepo, você pode adicionar a opção `-r` para atualizar recursivamente:

```bash
npx taze --include /rsbuild/ -w -r
```

- Adicione `-l` para atualizar versões bloqueadas:

```bash
npx taze --include /rsbuild/ -w -l
```

- Para atualizar para uma versão principal:

```bash
npx taze major --include /rsbuild/ -w
```

> Para mais opções, consulte a [documentação do taze](https://github.com/antfu-collective/taze).
