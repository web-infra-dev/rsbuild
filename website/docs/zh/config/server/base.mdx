# server.base

- **类型：** `string`

- **默认值：** `/`

`server.base` 用于配置服务端的[基础路径](/guide/basic/server#服务端基础路径)。

### 示例

默认情况下，服务端的基础路径为 `/`，用户可通过 http://localhost:3000 访问到 `index.html`。

当我们希望通过 http://localhost:3000/foo 访问到 `index.html` 时，可将 `server.base` 修改为 `/foo`。

```ts
export default {
  server: {
    base: '/foo',
  },
};
```
