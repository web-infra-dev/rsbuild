# performance.profile

- **类型：** `boolean`
- **默认值：** `false`

是否捕获每个模块的耗时信息，对应 Rspack 的 [profile](https://rspack.rs/zh/config/other-options#profile) 配置。

开启后：

- Rsbuild 会通过 [bundle analyzer](https://github.com/webpack-contrib/webpack-bundle-analyzer) 自动生成 `dist/stats.json` 文件。
- Rspack 在生成 stats.json 等统计文件时，会将构建的耗时信息也包含进去。

## 示例

```ts title="rsbuild.config.ts"
export default {
  performance: {
    profile: true,
  },
};
```

## 指南

请参考 [构建性能分析](/guide/debug/build-profiling) 页面来了解更多分析构建性能方法。
