# dev.beforeStartUrl

- **类型：** `() => Promise<void> | void`
- **默认值：** `undefined`

`dev.beforeStartUrl` 用于在打开 `startUrl` 前执行一段回调函数，该配置项需要与 [server.open](/config/server/open) 一同使用。

```js
export default {
  server: {
    open: true,
  },
  dev: {
    beforeStartUrl: async () => {
      await doSomeThing();
    },
  },
};
```
