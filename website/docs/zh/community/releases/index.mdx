# 总览

## 更新日志

请访问 [GitHub - release](https://github.com/web-infra-dev/rsbuild/releases) 来查看 Rsbuild 每个版本的变更内容。

## 语义化版本

Rsbuild 遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 语义化版本规范。

- 主版本号：包含不兼容的 API 变更。
- 次版本号：包含向下兼容的功能性变更。
- 修订号：包含向下兼容的问题修正。

## 发版周期

- Rsbuild 通常会在每周发布若干个 patch 版本。
