Called before starting the production preview server.

Use the `server` parameter to access the preview server and register custom middlewares.

- **Type:**

```ts
type MaybePromise<T> = T | Promise<T>;

type OnBeforeStartProdServerFn = (params: {
  /**
   * The preview server instance.
   */
  server: RsbuildPreviewServer;
  /**
   * Context information for all environments.
   */
  environments: Record<string, EnvironmentContext>;
}) => MaybePromise<void>;

function OnBeforeStartProdServer(callback: OnBeforeStartProdServerFn): void;
```
