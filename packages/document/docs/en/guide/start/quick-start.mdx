## Command Line Interface

Rsbuild comes with a lightweight CLI that includes commands such as `dev` and `build`.

```json
{
  "scripts": {
    // starting the dev server
    "dev": "rsbuild dev",
    // build the app for production
    "build": "rsbuild build",
    // preview the production build locally
    "preview": "rsbuild preview"
  }
}
```

Refer to the [CLI Tool](/guide/basic/cli) to learn about all available commands and options.

## Build Entry

By default, Rsbuild CLI uses `src/index.(js|ts|jsx|tsx)` as the build entry. You can modify the build entry using the `source.entries` configuration option.

- **Type:**

```ts
type Entries = Record<string, string>;
```

- **Default:**

```ts
const defaultEntry = {
  index: 'src/index.(js|ts|jsx|tsx)',
};
```

- **Example:**

```ts title="rsbuild.config.ts"
import { defineConfig } from '@rsbuild/core';

export default defineConfig({
  source: {
    entries: {
      foo: './src/pages/foo/index.ts',
      bar: './src/pages/bar/index.ts',
    },
  },
});
```
