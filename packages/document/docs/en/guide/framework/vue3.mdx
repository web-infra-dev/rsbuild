# Build Vue 3 App

Although the Modern.js framework is built on React, the underlying Rsbuild is not coupled with React. Therefore, you can use Rsbuild to build your Vue applications and leverage most of the capabilities provided by Rsbuild.

In this document, you will learn how to build a Vue 3 application using Rsbuild.

## Using Vue Plugin

To compile Vue SFC (Single File Components) and JSX syntax, you need to register the Rsbuild [Vue plugin](/plugins/list/plugin-vue). The plugin will automatically add the necessary configuration for Vue builds and remove the built-in React-related configurations.

For example, register in `rsbuild.config.ts`:

```ts title="rsbuild.config.ts"
import { defineConfig } from '@rsbuild/cli';
import { pluginVue } from '@rsbuild/plugin-vue';

export default defineConfig({
  plugins: [pluginVue()],
});
```

## Type Definitions

In a TypeScript project, you need to add type definitions for `*.vue` files so that TypeScript can recognize them correctly.

Create `env.d.ts` in the `src` directory and add the following content:

```ts title="src/env.d.ts"
declare module '*.vue' {
  import type { DefineComponent } from 'vue';

  // eslint-disable-next-line @typescript-eslint/ban-types
  const component: DefineComponent<{}, {}, any>;
  export default component;
}
```
