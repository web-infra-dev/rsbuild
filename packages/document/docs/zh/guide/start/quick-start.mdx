## 使用 Rsbuild CLI 工具

Rsbuild 提供了一个轻量的 CLI 工具，包含 dev、build 等基础命令，它主要用于构建非 React 项目。

如果你的项目不是基于 React 的，比如开发一个 Vue 项目，那么你可以使用 Rsbuild CLI 工具来构建你的项目。

请参考 [使用 Rsbuild CLI](/guide/basic/cli) 来了解相关用法。
