# Solid

在这篇文档中，你可以了解到如何基于 Rsbuild 来构建一个 Solid 应用。

## 创建 Solid 项目

你可以使用 `create-rsbuild` 来创建一个 Rsbuild + Solid 的项目，调用以下命令：

import { PackageManagerTabs } from '@theme';

<PackageManagerTabs command="create rsbuild@latest" />

然后在 `Select framework` 时选择 `Solid` 即可。
