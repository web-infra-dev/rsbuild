# server.host

- **类型：** `string`
- **默认值：** `0.0.0.0`

指定 Rsbuild Server 启动时监听的 host。

默认情况下，Rsbuild Server 会监听 `0.0.0.0`，这代表监听所有 IPv4 的网络接口，包括 `localhost` 和公网地址。

你可以使用 `server.host` 配置或 `--host` 参数来指定 host (`--host` 参数的优先级高于 `server.host` 配置)。

如果你希望 Rsbuild Server 只监听 `localhost`，可以设置为：

```ts
export default {
  server: {
    host: 'localhost',
  },
};
```

## IPv6 支持

如果你希望监听所有 IPv6 的网络接口，可以设置为：

```ts
export default {
  server: {
    host: '::',
  },
};
```

如果你希望监听 IPv6 指定的 host，可以设置为：

```ts
export default {
  server: {
    host: '::1',
  },
};
```

此时，你可以通过 `http://[::1]:8080/` 访问页面。
