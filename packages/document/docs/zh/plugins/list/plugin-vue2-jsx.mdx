# Vue2 JSX 插件

Vue2 JSX 插件提供了对 Vue 2 JSX / TSX 语法的支持，插件内部集成了 [@vue/babel-preset-jsx](https://github.com/vuejs/jsx-vue2)。

:::tip
在使用 Rspack 打包时，使用 Vue2 JSX 插件将会显著降低构建性能。因为 Rspack 默认使用的是更快的 SWC 编译，而 Vue2 JSX 插件依赖更慢的 Babel 编译，产生了额外的编译开销。
:::

## 快速开始

### 安装插件

你可以通过如下的命令安装插件:

import { PackageManagerTabs } from '@theme';

<PackageManagerTabs command="add @rsbuild/plugin-vue2-jsx -D" />

### 注册插件

你可以在 `rsbuild.config.ts` 文件中注册插件：

```ts title="rsbuild.config.ts"
import { pluginVue2Jsx } from '@rsbuild/plugin-vue2-jsx';

export default {
  plugins: [pluginVue2Jsx()],
};
```

如果你使用了 Rsbuild 的 JavaScript API，可以通过 [addPlugins](/api/javascript-api/instance.html#rsbuildaddplugins) 方法来注册插件。

注册完插件后，你可以直接在代码中可以使用 Vue 的 [JSX / TSX 语法](https://github.com/vuejs/jsx-vue2)。

## 配置

如果你需要自定义 Vue JSX 的编译行为，可以使用以下配置项。

### vueJsxOptions

传递给 `@vue/babel-preset-jsx` 的选项，请查阅 [@vue/babel-preset-jsx 文档](https://github.com/vuejs/jsx-vue2) 来了解具体用法。

- **类型：**

```ts
type VueJSXPresetOptions = {
  compositionAPI?: boolean | string;
  functional?: boolean;
  injectH?: boolean;
  vModel?: boolean;
  vOn?: boolean;
};
```

- **默认值：**

```ts
const defaultOptions = {
  injectH: true,
};
```

- **示例：**

```ts
pluginVue2Jsx({
  vueJsxOptions: {
    injectH: false,
  },
});
```
