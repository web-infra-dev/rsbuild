# Rsbuild Types

本章节描述了 Rsbuild 提供的一些类型定义。

## BuilderInstance

Rsbuild 实例的类型。

```ts
import type { BuilderInstance } from '@rsbuild/core';

let builder: BuilderInstance;
```

你可以通过泛型来传入 Provider 的类型，使 Rsbuild 实例获得更准确的类型定义：

```ts
import type { createRsbuild, type BuilderInstance } from '@rsbuild/core';
import type { BuilderWebpackProvider } from '@rsbuild/webpack-provider';

let builder: BuilderInstance<BuilderWebpackProvider>;
```

## BuilderContext

Rsbuild 实例中 context 属性的类型定义。

```ts
import type { BuilderContext } from '@rsbuild/core';

const context: BuilderContext = builder.context;
```

## BuilderPlugin

Rsbuild 插件的类型，需要配合 provider 中提供的 `BuilderPluginAPI` 类型来使用。

```ts
import type { BuilderPlugin } from '@rsbuild/core';
import type { BuilderPluginAPI } from '@rsbuild/webpack-provider';

const myPlugin: BuilderPlugin<BuilderPluginAPI> = {
  name: 'my-plugin',
  setup() {},
};
```

## BuilderTarget

Rsbuild 构建产物的类型。

```ts
import type { BuilderTarget } from '@rsbuild/core';
```

## BuilderEntry

对应 `createRsbuild` 方法的 `entry` 选项的类型。

```ts
import type { BuilderEntry } from '@rsbuild/core';
```

## CreateBuilderOptions

对应 `createRsbuild` 方法的入参类型。

```ts
import type { CreateBuilderOptions } from '@rsbuild/core';
```
